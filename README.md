# Digital Tools Buying Website

This Product provide digital services and boost up workflow.Here user can buy their digital tool for any purpose with a cheapest price.Digital tool mainly work on reduce user time to create those thing with taking a lot of time.It's provide some digital tools with ready to use any time.

## Features

**Product Features**

- User can buy tool with " Buy Now" button
- When a user click on "Buy Now" button , the button will turn into "Added to Cart", And also generating a toast message with the name of the product.
- And when a user click on "Buy Now" button, the cart button will increase the cart list item, and also increase the Navbar cart icons value.

**Cart Features**

- When a user click on "Cart" button, this interface show a list of cart items.
- User can also remove the cart items with click on "Remove" button, which is included on cart items. And also generating a toast message with the cart items name.
- When user remove any items by click on "Remove" Button , it will decrease the cart list item value , total value and cart icons value top of the navbar.

### Technologies Used

- React.js
- Tailwind CSS, DaisyUI
- JavaScript (ES6+)
- React-Toastify (NPM Package)
- JSON (for product data)
